# Search Components

Styles for a basic search input field and result items are available for use:

- [SearchInput](?path=/docs/components-search-searchinput--default)
- [SearchResultCard](?path=/docs/components-search-searchresultcard--standard-result)

If your project requires search functionality, you will need to build it into your implementation of the above components. Explorer-1 does not provide any search functionality.

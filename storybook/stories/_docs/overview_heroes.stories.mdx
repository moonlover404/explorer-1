# Hero Components

Hero components are reserved for usage at the top of the page. There should only be one hero per page.

- [HeroFeature](?path=/docs/components-blocks-heroes-herofeature--default): Includes a text overlay.
- [HeroFocalPoint](?path=/docs/components-blocks-heroes-herofocalpoint--hero): Includes a large text overlay that works best with background images that have a focal point on the right side of the image.
- [HeroMedia](?path=/docs/components-blocks-heroes-heromedia--hero): No text overlay, just an image or video with a caption below.
